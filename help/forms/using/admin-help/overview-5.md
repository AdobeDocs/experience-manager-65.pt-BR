# Introdução ao trabalho com o Gerador de PDF {#introduction-to-working-with-pdf-generator}

PDF Generator converts various file formats to PDF. Também converte PDF em outros formatos de arquivo e otimiza o tamanho dos documentos PDF. For a list of supported file formats, see [Software support for PDF Generator](/help/forms/using/aem-forms-jee-supported-platforms.md)

**Sending files to PDF Generator for processing**

Há três maneiras de enviar arquivos para o Gerador de PDF para processamento:

* Administrators can access the PDFG pages in Administration Console. (See [Converting files using PDF Generator](/help/forms/using/admin-help/converting-files-using-pdf-generator.md).)
* Users can access the PDFG end-user pages by logging in to `http(s)://'[server]:[port]'/pdfgui.` From there, they can access the PDFG Network Printer, Create PDF, HTML to PDF, Export PDF, and Optimize PDF pages.
* Você pode configurar pontos de extremidade para os serviços. Consulte <!--Fix broken link to Managing Endpoints --> [Gerar recomendações](configuring-watched-folder-endpoints.md#generate-pdf-service-recommendations)do serviço PDF.
